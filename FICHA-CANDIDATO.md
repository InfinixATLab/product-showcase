
- Seção 1: Instruções para rodar

1- Quais variáveis de ambiente são necessárias?
Nenhuma variável de ambiente, pois este projeto consume uma API PÚBLICA 

1- Como instalar dependências?
com `npm install`

1- Como rodar o projeto?
com `npm run dev`


- Seção 2: Decisões de design

1- Por que você escolheu essa estrutura de pastas?
Escolhi essa estrutura de pastas foi pensada para manter o projeto organizado, escalável e fácil para outros entenderem e conseguirem se adaptar com mais facilidade.

2- Qual foi a maior dificuldade que você encontrou e como superou?
Eventualmente tive dificuldade em lembrar o nome exato de alguns métodos e atributos, recorrendo pontualmente à documentação para confirmar a sintaxe correta.

3- O que você não teve tempo de fazer (dentro do timebox) e como você faria se tivesse mais tempo?
Tive tempo para apresentar uma amostra do que sou capaz de desenvolver, demonstrando que um básico bem feito pode ser sólido e eficiente.

Seção final: Recomendações
1- Refatorar código para componentes mais reutilizáveis, evitando duplicação.
2- Implementar testes unitários e integração para serviços e componentes.
